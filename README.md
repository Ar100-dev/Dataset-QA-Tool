# Dataset QA Tool

A professional desktop application for verifying YOLO object detection datasets.

## Features

✔ Dataset structure validation

✔ Missing image-label detection

✔ Duplicate image detection

✔ Corrupt image detection

✔ Invalid YOLO label detection

✔ Class distribution visualization

✔ Auto Fix

✔ Supports

- Standard YOLO
- Roboflow YOLO Export

## Installation

pip install -r requirements.txt

python main.py
